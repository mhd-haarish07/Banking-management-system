# Bank Management System

A Bank Management System designed to simplify and automate core banking operations 
such as account creation, deposits, withdrawals, balance inquiries, and transaction 
history — built to demonstrate database management and application logic.

## Features
- 👤 Create, update, and delete customer accounts
- 💰 Deposit and withdraw funds
- 📊 Check account balance
- 🧾 View transaction history
- 🔐 Secure login/authentication for users and/or admin
- 🗃️ Data stored and managed using a database

## Tech Stack
- **Language:** Java 
- **Database:** MySQL 
- **Interface:** Console-based 
- **Other Tools:** Git, GitHub
